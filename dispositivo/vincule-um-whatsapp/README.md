# Vincule um Whatsapp

## Como funciona?

O processo de integração é igual a um Whatsapp Web.

## Primeiros Passos

Acesse a plataforma Wavoip e selecione o dispositivo que você gostaria de vincular seu número de Whatsapp. Siga as instruções disponíveis na tela para finalizar o vínculo.&#x20;

## Como enviar somente o QRCode para o cliente

Na página do dispositivo, passando o mouse por cima do QRCode você terá duas opções: abrir o QR em uma nova guia, ou copiar o link diretamente. Você também pode gerar esse link no seguinte formato:

```
https://devices.wavoip.com/{token}/whatsapp/qr-image
```

Esse link vai exibir um resultado semelhante à imagem abaixo:

<figure><img src="../../.gitbook/assets/Screenshot_4.png" alt=""><figcaption></figcaption></figure>

## É possível vincular um número sem QRCode?

A integração por código OTP (Sem QRCode) ainda não está disponível.

# Whatsapp Externo

## O que é o Whatsapp Externo?

A integração Whatsapp externo permite que você compartilhe a sua sessão de API com a Wavoip, permitindo assim que seu cliente não precise vincular o QRCode direto no Wavoip.

## Como funciona?

Se você já trabalha com alguma API baseada em WhatsApp Web/Desktop, a Wavoip faz uma ponte com seu sistema. Dessa forma, toda comunicação com servidor do Whatsapp passa pela sua infraestrutura, eliminando a necessidade de escanear o número novamente.



## Quando usar?

Você deve usar essa integração quando já trabalha com uma API baseada em Whatsapp Web/Desktop (Baileys, Whatsmeow...) e gostaria de reaproveitar a sessão.

Também deve utilizar caso queira assumir o controle da conexão em sua infraestrutura.



## Quando não usar?

Não é recomendado utilizar quando falta conhecimento técnico sobre as bibliotecas e protocolos relacionados a API do Whatsapp Web/Desktop.



## Quais APIs de WhatsApp consigo compartilhar?

Atualmente, as integrações disponiveis são:

* Evolution
* Baileys

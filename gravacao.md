# Gravação

## Como baixar uma gravação?

Você pode acessar o link de download da gravação direto pelo nosso painel ou pode montar e acessar via API a estrutura do link da gravação é&#x20;

```
https://storage.wavoip.com/{WHATSAPP_CALL_ID}
```

Substitua {WHATSAPP\_CALL\_ID} pelo ID da ligação. As gravações podem estar indisponíveis por alguns minutos após a ligação ter finalizado. Aguarde alguns minutos e tente novamente.



## Quanto tempo a gravação fica disponível?

Não existe um tempo limite para ligação ser removida na Wavoip, a gravação será removido apenas quando solicitado.
